# growth_analysis
## 项目背景
- 实现对modis数据的检索下载，并对下载下来的数据进行按天处理，然后计算NDVI植被指数
## 安装
- 本项目提供两种安装方式
1. 使用git将项目拉取到本地
```git
git clone https://github.com/sanliyang/growth_analysis.git
```
然后单击start.exe即可运行程序

2. 使用一键式安装引导程序安装
    - 将本项目中的setup文件夹下载到本地
    - 双击运行setup/setup.exe进行安装
